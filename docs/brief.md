# CampusConnect brief

## Goal
Build a full-stack campus/community SPA based on the wireframes.

## Required stack
- React frontend
- Express backend
- MongoDB + Mongoose
- SASS
- Auth + authorization
- Sessions + cookies
- PWA
- GitHub-ready repo
- Deployable to a live server with HTTPS

## Core screens
- Home
- Login / Signup
- Create or Edit Event/Post
- Dashboard / Manage Posts

## MVP features
- User signup, login, logout
- Protected dashboard
- Create, edit, delete own posts/events
- Form validation on client and server
- Responsive layout
- Installable PWA shell

## Suggested roles
- Standard user
- Admin later if needed

## Done when
- Local dev works
- App builds successfully
- README explains setup
- Environment variables are documented
